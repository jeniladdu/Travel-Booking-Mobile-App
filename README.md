# Travel Booking App

A clickable Travel Booking App prototype designed in Figma. The app allows users to explore destinations, book flights and hotels, manage reservations, and enjoy a smooth travel planning experience.

## Prototype Link

https://www.figma.com/proto/6PraaHUDk9vEsDp9T1Djg6/Travel-Booking?node-id=2-2&t=PoW2oggdW3pihLYY-1

## Features

* Destination Search
* Flight Booking
* Hotel Booking
* Travel Package Exploration
* Booking Management
* User-Friendly Interface

## Tools Used

* Figma
* UI/UX Design
* Interactive Prototyping
